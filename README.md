meteor-scheduler-app
===================

Demo of DHTMLX Scheduler with Meteor

### How to start

To run the app, do the next after cloning the repo

~~~
meteor npm install
meteor
~~~

after that, open in a browser http://localhost:3000 to check the scheduler
